# To Do Application

This project is for track your to Do's

## Installation

To use this project, follow these steps:

1. Clone the repository.
2. Cd client
3. Install dependencies with `npm install`.
4. Npm run dev

## Usage

To run the project, use the following command:

```bash
npm start
```
